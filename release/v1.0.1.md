# Release Notes - v1.0.1

This is a maintenance release that addresses CI pipeline failures and updates the internal version tracking.

## 🛠 Fixes & Improvements

- **CI Pipeline Stability:** 
  - Disabled automatic database migrations specifically for the `IntegrationTesting` environment to prevent connection errors when using the In-Memory database.
  - Re-enabled core database seeding (Role assignment) for all environments to ensure integration tests have access to the necessary roles (ADMIN, TECHNICIAN, etc.).
- **Health Check Resilience:** 
  - Relaxed the health check integration test to verify the presence of a version string and "Healthy" status rather than a hardcoded version number.
- **Version Bump:** 
  - Updated application version to `1.0.1` across frontend and backend configurations.
