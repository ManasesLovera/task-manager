# Release Notes - v1.0.2

This release fixes the build-time environment variable injection for the production frontend and updates the internal version tracking.

## 🛠 Fixes & Improvements

- **Frontend Build Injection:**
  - Modified the `Dockerfile` to use `ARG` and `ENV` for `VITE_API_URL`. This ensures that Vite correctly "bakes" the backend API URL into the Javascript files during the Docker build phase, preventing `undefined` errors in production.
- **Version Bump:**
  - Updated application version to `1.0.2` across frontend and backend configurations.
